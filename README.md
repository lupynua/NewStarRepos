# NewStarRepos
Android application that allows to explore the list of the most starred Github repositories created during last month.

MVVM, Clean Architecture, Jetpack Compose

Libraries:
* Jetpack Compose
* Retrofit(OkHTTPClient)
* Koin
* Coil
